# python-web-scraper
Python Web Scraper Tool
Project Overview

This project is a Python-based Web Scraper that extracts quotes and author names from a website and saves the data into a CSV file. The scraper automatically navigates through multiple pages and collects all available data.

This project demonstrates web scraping, data extraction, automation, and GUI development using Python.

## Features

Extract quotes and author names from webpages
Automatically scrape multiple pages
Add request headers to avoid blocking
Add delay between requests to simulate human browsing
Error handling for network failures
Export scraped data to CSV file
Simple GUI interface for users
